## 🚀 БЫСТРЫЙ СТАРТ (Рекомендуется)

### Шаг 1️⃣: Build
```bash
npm run build
```

### Шаг 2️⃣: Открыть Netlify
Перейти на: **https://app.netlify.com/**

### Шаг 3️⃣: Drag & Drop
Перетащить папку **`dist`** в браузер

```
     📁 dist
       ⬇️  (перетащить сюда)
   ┌────────────────────┐
   │                    │
   │   Netlify Drop     │
   │                    │
   └────────────────────┘
```

**🎉 ГОТОВО! Ваш сайт онлайн!**

Получите ссылку: `https://random-name.netlify.app`

## 💻 С ПОМОЩЬЮ ТЕРМИНАЛА

### Вариант А: Автоматический скрипт
```bash
./deploy.sh
```

### Вариант Б: Netlify CLI
```bash
# Установка (один раз)
npm install -g netlify-cli

# Авторизация (один раз)
netlify login

# Деплой
npm run build
netlify deploy --prod
```

## 🔥 САМЫЕ ЧАСТЫЕ ВОПРОСЫ

### Q: Нужен ли аккаунт Netlify?
**A:** Да, но регистрация бесплатная (можно через GitHub)

### Q: Сколько стоит?
**A:** Бесплатно! (100 GB bandwidth, 300 build minutes/месяц)

### Q: Можно ли изменить URL?
**A:** Да! Site settings → Domain management → Change site name

### Q: Как обновить сайт?
**A:** Просто повторите шаги (новый build + drag & drop)

### Q: Что делать если что-то не работает?
**A:** 
1. Проверьте консоль браузера (F12)
2. Убедитесь что локально всё работает: `npm run preview`
3. Проверьте логи деплоя в Netlify dashboard

## ✅ Проверка после деплоя

Откройте ваш сайт и проверьте:

```
✓ Страница загружается
✓ Видны блоки слева
✓ Блоки цветные (синий, фиолетовый, оранжевый, желтый)
✓ Drag & drop работает
✓ Спрайт виден
✓ Кнопка "Запустить" работает
✓ Спрайт двигается при выполнении
✓ Костюмы меняются
```
